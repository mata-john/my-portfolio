# Project Overview

This portfolio is a production-ready web application that showcases my software engineering experience, projects, development workflow, and technical decision-making. It is intended to present both practical implementation work and the approach used to build and improve the application over time.

# Purpose

The project was built to:

- showcase real internship and academic experience
- demonstrate frontend and full-stack development skills
- practice modern software engineering principles
- provide recruiters and developers with a clear overview of my work

# Target Audience

- Recruiters
- Engineering Managers
- Software Developers
- Technical Interviewers

# Key Features

- Responsive design
- Component-based architecture
- Modern UI built with Tailwind CSS
- TypeScript support
- Accessibility-first development
- Professional project showcase
- Resume download
- Behind the Build engineering documentation

# Project Structure

The repository is organized into the following main directories:

- `app/` - application routes and page-level structure
- `components/` - reusable UI components
- `public/` - static assets such as images and documents
- `docs/` - project documentation

# Getting Started

Clone the repository and install dependencies:

```bash
git clone <repository-url>
npm install
npm run dev
```

# Production Build

To create a production build:

```bash
npm run build
npm start
```
